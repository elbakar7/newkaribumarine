# Karibu Marine - Luxury Boat Tours in Zanzibar

A premium WordPress theme for Karibu Marine, offering luxury boat tours, private charters, and bespoke island experiences in Zanzibar.

## About Karibu Marine

Karibu Marine provides gentle, welcoming boat journeys across Zanzibar's calm blue waters. We offer boutique charters, thoughtful service, and crewed experiences crafted for calm journeys across Stone Town and the surrounding isles.

## Our Tours

- **Prison Island Tour** - Unhurried visit to meet Aldabra tortoises and explore colonial history
- **Nakupenda Sandbank** - Soft cruise to a turquoise sandbank with freshly prepared seafood
- **Bawe Island Experience** - Full-day island escape with vibrant reefs and attentive guides
- **Chumbe Island Sanctuary** - Conservation-focused cruise with marine rangers
- **Private Charters** - Fully customizable experiences tailored to your preferences
- **Full Day Charters** - Extended voyages with curated meals and island stops

## Theme Installation

1. Upload the theme folder to `/wp-content/themes/`
2. Activate the theme in WordPress Admin
3. Customize colors, fonts, and layouts via the Theme Customizer
4. Use the pre-built blocks to create your pages
